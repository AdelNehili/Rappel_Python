# Difficultés fréquentes en Python

Les élèves rencontrent souvent des difficultés avec les notions suivantes :

## Gestion des fonctions
- Comprendre l’intérêt de `return`
- Différencier `yield` et `return`

## Parcours de structures de données
- Comprendre ce que signifie parcourir une liste
  - via les indices
  - via les éléments

## Structures de données
- Comprendre la différence fondamentale entre :
  - `list`
  - `tuple`
  - `dict`

## Manipulation de fichiers
- Ouvrir un fichier texte avec `open(file, open_mode)`

## Opérations bitwise
- AND
- OR
- XOR
- Left shift
- Right shift
